Cloud Sync Settings

Paste your Google Apps Script Web App URL here and click "Save URL". After that, every new/edited worker card will automatically save to Google Sheet and Google Drive (photo) as well — data on this website (this browser) will remain the same.

How it works

1. When you Save a Card, data is instantly saved in this browser (localStorage), and also sent to Google Sheet + Drive (if URL is connected).
2. Clicking "Pull from Sheet" fetches records present in the Sheet into this website (workers added from other devices/browsers will also appear).
3. If there is no internet or cloud sync fails, your data remains safe in this browser — no data loss.
4. Steps for setting up the Apps Script (Code.gs) file are provided in chat.
